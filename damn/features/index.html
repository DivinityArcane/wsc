wsc.dAmn » Features

wsc.dAmn is an example of wsc in use, and also acts as an alternative to the official dAmn client on deviantART. The live demo provides the existing features of the official client, extra features implemented by superdAmn, and adds even more features for you to use. This page is an attempt to detail the available features.
