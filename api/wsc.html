    wsc

    This function is the plugin method allowing you to run wsc using jQuery. The client is designed to work with jQuery, but the objects are abstracted out of this file/function to make it easier to maintain and such.

    This function generates a new wsc_client object if there is no client present in the window. Apart from that, this function can be used to invoke functions on the client object.

    (jQuery) wsc(method(String) Name of the method to call on the client object. , options(Object) Use this to pass arguments to the method being called. )

    $.fn.wsc = function( method, options ) { client = $(window).data('wscclient'); if( method == 'init' || client === undefined ) { if( client == undefined ) { client = wsc_client( $(this), options, $.browser.mozilla ); $(window).resize(client.resizeUI); $(window).focus(function( ) { client.control.focus(); }); } $(window).data('wscclient', client); } if( method != 'init' && method != undefined ) { pieces = method.split('.'); o = client; for( i in pieces ) { p = pieces[i]; if( p in o ) o = o[p]; else return this; } o( $(this), options); } return this; };
