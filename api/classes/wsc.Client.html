wsc.Client Class

Defined in: src/client.js:1

Module: wsc

Available since 0.0.1

An entire chat client. Instances of this object orchestrate the operation of the client. Other objects are loaded in to control different parts of the client. These components can be reasonably swapped out, assuming they provide the same functionality.
