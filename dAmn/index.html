wsc.dAmn

wsc is a chat client for dAmn-like chat servers. wsc.dAmn is a demo of the chat client that connects to dAmn via WebSockets. This client uses HTML, CSS, and Javascript. By default, it does not use Flash.

In comparison to the official web client for dAmn; wsc has several useful extra features, is configurable, and gives you custom emoticons from the emote cloud.

Control

Above the input line of the client, there are a few buttons. The multiline input button, the Autojoin button, and the emote picker button.

To join all of your autojoin channels at once, click the autojoin button. Clicking the emote picker button brings up a small popup listing the available emotes.
